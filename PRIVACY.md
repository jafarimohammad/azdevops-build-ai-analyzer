# Privacy Policy

**Azure DevOps Build AI Analyzer** ("the Extension") is an open-source Azure DevOps extension.

## Data Collection

The Extension does **not** collect, store, or transmit any personal data to the extension author or any third-party service.

## Data Processing

When a pipeline build fails, the Extension reads the build log from your Azure DevOps instance and sends it to the LLM endpoint **you configure** in the pipeline task parameters. That endpoint is entirely under your control (self-hosted or your own cloud subscription). No data passes through any server operated by the extension author.

## Third-Party Services

The only external service involved is the LLM endpoint you supply. Please review the privacy policy of that service separately.

## Changes

This policy may be updated alongside new versions of the Extension. Changes are tracked in the repository commit history.
